--------テーブル構成-------

テーブル名 : cross_tag_info
要素 : tag_id, tag_name, cross_name, status, direction
説明 : タグのIDと名前および所属交差点名、タグの役割、方角を保存
（タグid, タグ名, 所属交差点名, タグの役割, タグのある方角）


テーブル名 : road_tag_info
要素 : tag_id, cross_name_1, cross_name_2
説明 : 交差点間のタグから接続する交差点
（タグID, 行き先の交差点名, 来し方の交差点名）


テーブル名 : cross_position
要素 : cross_name, x, y
説明 : 交差点の座標保存
（交差点名, x座標, y座標）


テーブル名 : road_info
要素 : cross_name_1, cross_name_2, dist, oneway
説明 : 交差点間の距離と通行方向
（交差点名1, 交差点名2, 距離, 通行方向）


テーブル名 : control
要素 : car_id, cross_name, tag_id, origin, destination, status, time
説明 : 交差点での車の状態管理
（交差点名, 車ID, 接続したタグID, 来し方, 行き先, 車の状態, 時間） 


--------規則一覧--------

[通行方向] oneway
    0: 双方向
    1: 交差点1 -> 交差点2
    2: 交差点2 -> 交差点1

[来し方] origin
    0: 北
    1: 東
    2: 南
    3: 西

[行き先] destination
    0: Uターン
    1: 左折
    2: 直進
    3: 右折

[方角] direction
    0 : 北
    1 : 東
    2 : 南
    3 : 西

[状態] status
    connect : 接続開始
    stop    : 停止
    entry   : 進入中
    passed  : 通過済

[交差点名] cross_name
cross_(nnn)

[タグ名] tag_name
tag_(direction)_(status)_(nnn)

[タグID] tag_id
tag_(direction)_(status)_(nnn)_id

[距離] dist
単位     : cm
有効桁数　: 小数点第2位

[座標] x, y
単位     : cm
有効桁数　: 小数点第2位
