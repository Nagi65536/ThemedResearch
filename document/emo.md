# 様々な記録

### RC522と磁場強度について

|  磁石              |  読み取り  |  測定可能距離  |
| :---------------- | :-------: | :----------: |
|  マグネットテープ    |  可能     |  変化なし      |
|  ネオジウム（小）    |  可能     |  変化なし      |
|  ネオジウム（最強）  |  不可     |  x            |

### RC522と検知可能距離について
* 最大 2.5cm ~ 3.0cm（電池2個3.0Vの場合）
* 最大 1.6cm ~ 3.7cm（esp32からの場合）
* 雑誌等は検知可能距離に影響しない
* 磁石やesp32等の電磁場に干渉するものは検知可能距離が短くなる、もしくは検知不可能となる
* 上記の場合、磁石等を離した後も検知不可能となる場合もある
